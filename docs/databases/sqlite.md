[SQLite](https://www.sqlite.org/) is supported by continuousphp containers.

## Specification 

Our containers run SQLite 3 with the following PHP extensions:

* pdo_sqlite
* sqlite3

## Connecting to SQLite 

There are no Authentication settings.
